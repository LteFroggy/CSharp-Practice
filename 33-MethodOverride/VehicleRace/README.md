# VehicleRace

다양한 탈것들이 경주하는 프로그램을 작성하시오.

**요구사항**

- `Vehicle` 클래스를 만들고, `Name` 속성과 `Speed` 속성을 가짐
- `Vehicle` 클래스에 `virtual` 메서드 `Move()`를 만들고, "[이름]이(가) 이동합니다. 속도: [속도]km/h"를 출력함
- `Car` 클래스는 `Vehicle`을 상속받고, 기본 속도는 120, `Move()`를 오버라이드하여 "[이름]이(가) 도로를 달립니다! 속도: [속도]km/h"를 출력함
- `Airplane` 클래스는 `Vehicle`을 상속받고, 기본 속도는 900, `Move()`를 오버라이드하여 "[이름]이(가) 하늘을 납니다! 속도: [속도]km/h"를 출력함
- `Boat` 클래스는 `Vehicle`을 상속받고, 기본 속도는 60, `Move()`를 오버라이드하여 "[이름]이(가) 바다를 항해합니다! 속도: [속도]km/h"를 출력함
- `Vehicle[]` 배열에 각 탈것을 담고, `foreach`로 순회하며 `Move()` 호출

## 예상 실행 결과

```
=== 경주 시작! ===
스포츠카이(가) 도로를 달립니다! 속도: 120km/h
보잉747이(가) 하늘을 납니다! 속도: 900km/h
요트이(가) 바다를 항해합니다! 속도: 60km/h
```
