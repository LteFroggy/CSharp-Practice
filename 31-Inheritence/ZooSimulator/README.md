# 동물원 시뮬레이터

동물원을 시뮬레이션하는 프로그램을 작성하세요. 상속을 활용하여 다양한 동물 클래스를 구현합니다.

**요구사항**

1. `Animal` 부모 클래스 구현

   - 속성: `Name`(이름), `Age`(나이)
   - protected 필드: `_sound`(울음소리)
   - 메서드: `Eat()` - "먹이를 먹습니다" 출력
   - 메서드: `MakeSound()` - 각 동물의 울음소리 출력
   - 메서드: `DisplayInfo()` - 동물 정보 출력

2. `Animal`을 상속받는 3개의 동물 클래스 구현

   - `Lion` 클래스
     - 울음소리: "으르렁!"
     - 고유 메서드: `Hunt()` - "사냥을 합니다" 출력
   - `Elephant` 클래스
     - 울음소리: "뿌우!"
     - 고유 메서드: `SprayWater()` - "물을 뿌립니다" 출력
   - `Penguin` 클래스
     - 울음소리: "꽥꽥!"
     - 고유 메서드: `Swim()` - "수영을 합니다" 출력

3. 각 자식 클래스의 생성자에서 `base()` 키워드를 사용하여 부모 생성자 호출

4. 동물원에 동물 배열을 만들고 모든 동물의 정보와 울음소리를 출력

## 예상 실행 결과

```
=== 동물원에 오신 것을 환영합니다! ===

[동물 정보]
이름: 시바, 나이: 5살
이름: 덤보, 나이: 10살
이름: 뽀로로, 나이: 3살

[동물 소리]
시바: 으르렁!
덤보: 뿌우!
뽀로로: 꽥꽥!

[동물 행동]
시바이(가) 먹이를 먹습니다.
시바이(가) 사냥을 합니다.
덤보이(가) 먹이를 먹습니다.
덤보이(가) 물을 뿌립니다.
뽀로로이(가) 먹이를 먹습니다.
뽀로로이(가) 수영을 합니다.
```
