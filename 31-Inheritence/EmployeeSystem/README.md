# 직원 급여 시스템

회사의 직원 급여 시스템을 구현하세요. 상속을 활용하여 다양한 직종의 급여 계산을 처리합니다.

**요구사항**

1. `Employee` 기반 클래스 구현

   - protected 필드: `_name` (string), `_baseSalary` (int)
   - 생성자: 이름과 기본급을 받아 초기화
   - `GetName()` 메서드: 이름 반환
   - `GetBaseSalary()` 메서드: 기본급 반환
   - `CalculatePay()` 메서드: 기본급을 반환 (자식 클래스에서 숨기기 가능)
   - `PrintInfo()` 메서드: 직원 정보 출력

2. `Manager` 클래스 (`Employee` 상속)

   - 추가 필드: `_bonus` (int)
   - 생성자: 이름, 기본급, 보너스를 받음 (`base()`로 부모 생성자 호출)
   - `CalculatePay()`: 기본급 + 보너스 반환 (`new` 키워드로 부모 메서드 숨기기)
   - `PrintInfo()`: 기본 정보 + 보너스 정보 출력 (`new` 키워드로 부모 메서드 숨기기)

3. `Developer` 클래스 (`Employee` 상속)

   - 추가 필드: `_overtimeHours` (int)
   - 생성자: 이름, 기본급, 초과근무시간을 받음
   - `CalculatePay()`: 기본급 + 초과근무시간 * 20000 반환
   - `PrintInfo()`: 기본 정보 + 초과근무시간 출력

4. `Intern` 클래스 (`Employee` 상속)

   - 추가 필드 없음
   - 생성자: 이름만 받음 (기본급은 1500000으로 고정, `base()` 호출)
   - `CalculatePay()`: 기본급 그대로 반환
   - `PrintInfo()`: 기본 정보 출력

5. `Manager`, `Developer`, `Intern`을 각 1명씩 생성하고 `PrintInfo()`를 호출하여 정보 출력
