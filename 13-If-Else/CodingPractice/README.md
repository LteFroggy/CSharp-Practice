# 코딩 연습 - 제어문 소개 및 if/else 문

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

## 1. 순차문

정수형 변수 `kor`에 100, `eng`에 90을 저장하고, `tot`에 두 과목의 총점을, `avg`에 평균을 계산하여 저장한 후 출력하세요. 평균은 소수점 첫째 자리까지 표시합니다.

**실행 결과**

```
총점: 190
평균: 95.0
```

## 2. if 문: 조건이 참일 때 실행

### 2-1. 합격 판정

정수형 변수 `score`에 60을 저장하고, `score`가 60 이상이면 "합격"을 출력하세요.

**실행 결과**

```
합격
```

### 2-2. 다양한 비교 연산자 사용

정수형 변수 `x`에 10을 저장하고, `x`가 10과 같으면 "x는 10입니다."를 출력하고, `x`가 20이 아니면 "x는 20이 아닙니다."를 출력하세요.

**실행 결과**

```
x는 10입니다.
x는 20이 아닙니다.
```

### 2-3. 조건이 거짓인 경우

정수형 변수 `score`에 50을 저장하고, `score`가 60 이상이면 "합격"을 출력하세요. if 문 다음에 "프로그램 종료"를 출력하세요.

**실행 결과**

```
프로그램 종료
```

---

## 3. else 문: 조건이 거짓일 때 실행

### 3-1. 합격/불합격 판정

정수형 변수 `score`에 59를 저장하고, `score`가 60 이상이면 "합격", 그렇지 않으면 "불합격"을 출력하세요.

**실행 결과**

```
불합격
```

### 3-2. 상수 비교

`double` 상수 `k_Pi`에 3.14를 저장하고, `k_Pi`가 3.14와 같으면 "PI는 3.14입니다.", 그렇지 않으면 "PI는 3.14가 아닙니다."를 출력하세요.

**실행 결과**

```
PI는 3.14입니다.
```

---

## 4. else if 문: 여러 조건 처리

### 4-1. 두 수 비교

정수형 변수 `number1`에 10, `number2`에 20을 저장하고, `number1`이 더 크면 "number1이 더 큽니다.", `number2`가 더 크면 "number2가 더 큽니다.", 같으면 "둘 다 같습니다."를 출력하세요.

**실행 결과**

```
number2가 더 큽니다.
```

### 4-2. 양수/음수/0 판별

정수형 변수 `number`에 -10을 저장하고, 양수이면 "양수입니다.", 음수이면 "음수입니다.", 0이면 "0입니다."를 문자열 보간을 사용하여 출력하세요.

**실행 결과**

```
-10은 음수입니다.
```

### 4-3. 점수에 따른 학점 계산

정수형 변수 `score`에 85를 저장하고, `char` 변수 `grade`에 학점을 저장하세요. 90점 이상 A, 80점 이상 B, 70점 이상 C, 60점 이상 D, 그 미만은 F입니다. 점수와 학점을 출력하세요.

**실행 결과**

```
점수: 85점
학점: B
```

## 5. 중첩 if 문

### 5-1. 중첩 if 문 사용

문자열 변수 `name`에 "C#", 정수형 변수 `version`에 10을 저장하고, `name`이 "C#"이면 그 안에서 `version`이 9 이상인지 확인하여 최신 버전 메시지를 출력하세요.

**실행 결과**

```
C# 10은 최신 버전입니다.
```

위와 동일한 변수를 사용하되, 중첩 if 문 대신 `&&` 연산자를 사용하여 한 줄의 if 문으로 같은 결과를 출력하세요.

**실행 결과**

```
C# 10은 최신 버전입니다.
```

## 6. 논리 연산자와 if 문

### 6-1. AND 연산자 (&&)

정수형 변수 `number`에 1234를 저장하고, `number`가 1234와 같고 동시에 1000 이상인지 `&&` 연산자로 확인하여 "맞습니다."를 출력하세요.

**실행 결과**

```
맞습니다.
```

### 6-2. OR 연산자 (||)

정수형 변수 `number`에 1234를 저장하고, `number`가 1234와 같거나 또는 1000 이하인지 `||` 연산자로 확인하여 "하나라도 참이면 참"을 출력하세요.

**실행 결과**

```
하나라도 참이면 참
```

## 7. 짝수/홀수 판별 예제

정수형 변수 `a`에 15를 저장하고, 다음 세 가지 판별을 수행하세요:
- `[1]` if 문으로 짝수인지 확인하여 짝수이면 "짝수" 출력
- `[2]` if-else 문으로 홀수이면 "홀수", 짝수이면 "짝수" 출력
- `[3]` else if 문으로 3의 배수이면 "3의 배수", 5의 배수이면 "5의 배수", 7의 배수이면 "7의 배수", 아니면 "3, 5, 7의 배수가 아닌 수" 출력

**실행 결과**

```
홀수
3의 배수
```
