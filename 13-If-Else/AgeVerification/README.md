# 나이 확인 시스템

사용자의 나이에 따라 입장 가능 여부와 요금을 안내하는 프로그램을 작성하세요.

**요금 기준**

| 나이 | 상태 | 요금 |
|:---:|:---:|:---:|
| 0~6세 | 유아 | 무료 |
| 7~12세 | 어린이 | 5000원 |
| 13~18세 | 청소년 | 8000원 |
| 19세 이상 | 성인 | 12000원 |
| 음수 | 잘못된 입력 | - |

**요구사항**

- 나이 변수에 15를 저장함
- 나이가 0 미만이면 "잘못된 나이입니다."를 출력함
- 각 나이대에 맞는 상태와 요금을 출력함
- 조건에 맞는 하나의 메시지만 출력되어야 함

## 예상 실행 결과

```
나이: 15세
상태: 청소년
요금: 8000원
```
