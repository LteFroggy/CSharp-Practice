# TemperatureConverter

섭씨(Celsius) 온도를 화씨(Fahrenheit) 온도로 변환하는 프로그램을 만드시오.

**변환 공식**

```
화씨 = 섭씨 × 9 / 5 + 32
```

**요구사항**

- 섭씨 온도 값 3개(0, 25, 100)를 변환
- 결과는 소수점 첫째 자리까지 출력
- 출력 형식: `섭씨 [값]도 = 화씨 [값]도`

> 💡 **팁**: 정수끼리 나누면 소수점 이하가 버려지므로 주의

## 예상 실행 결과

```
섭씨 0도 = 화씨 32.0도
섭씨 25도 = 화씨 77.0도
섭씨 100도 = 화씨 212.0도
```
