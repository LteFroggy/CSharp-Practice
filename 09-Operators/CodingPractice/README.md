# 코딩 연습 - 연산자 개요 및 단항·산술 연산자

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 연산자와 피연산자

덧셈, 뺄셈, 곱셈, 나눗셈 식의 결과를 각각 `Console.WriteLine`으로 출력하세요. `3 + 5`, `3 - 5`, `3 * 5`, `3 / 5`를 계산합니다.

**실행 결과**

```
8
-2
15
0
```

---

## 2. 단항 연산자

### 2-1. 양수 연산자 (+)

정수형 변수 `value`를 선언하고 8을 저장한 후, `+value`를 대입하고 출력하세요.

**실행 결과**

```
8
```

### 2-2. 음수 연산자 (-)

정수형 변수 `value`에 8을 저장하고 `-value`를 대입하여 출력하세요. 그 다음 `value`에 -8을 저장하고 다시 `-value`를 대입하여 출력하세요.

**실행 결과**

```
-8
8
```

---

## 3. 변환 연산자 (캐스팅)

실수 `3.14`를 캐스팅 연산자 `(int)`로 정수로 변환하여 변수에 저장하고 출력하세요.

**실행 결과**

```
3
```

---

## 4. 덧셈 연산자 (+)

정수형 변수 `i`에 10, `j`에 20을 저장하고, 두 값을 더한 결과를 변수 `k`에 저장하여 출력하세요.

**실행 결과**

```
30
```

---

## 5. 뺄셈 연산자 (-)

`decimal` 형식 변수 `i`에 `3.14M`, `j`에 `3.14M`을 저장하고, `i - j`의 결과를 변수 `k`에 저장하여 출력하세요.

**실행 결과**

```
0.00
```

---

## 6. 곱셈 연산자 (*)

`long` 형식 변수 `i`에 100, `j`에 200을 저장하고, `i * j`의 결과를 변수 `k`에 저장하여 출력하세요.

**실행 결과**

```
20000
```

---

## 7. 나눗셈 연산자 (/)

### 7-1. 실수 나눗셈

`double` 형식 변수 `i`에 `1.5`, `j`에 `0.5`를 저장하고, `i / j`의 결과를 변수 `k`에 저장하여 출력하세요.

**실행 결과**

```
3
```

### 7-2. 정수 나눗셈과 실수 나눗셈 비교

정수형 변수 `a`에 5, `b`에 3을 저장하고, `a / b`(정수 나눗셈)와 `5.0 / 3`(실수 나눗셈)을 각각 출력하세요.

**실행 결과**

```
1
1.6666666666666667
```

---

## 8. 나머지 연산자 (%)

### 8-1. 나머지 구하기

정수형 변수 `first`에 5, `second`에 3을 저장하고, `first % second`의 결과를 변수 `result`에 저장하여 출력하세요.

**실행 결과**

```
2
```

### 8-2. 짝수 판별

정수형 변수 `number`에 10을 저장하고, `number % 2 == 0` 조건으로 짝수인지 판별하여 "짝수입니다"를 출력하세요.

**실행 결과**

```
짝수입니다
```

---

## 9. 산술 연산자 종합

정수형 변수 `a`에 5, `b`에 3을 저장하고, 덧셈, 뺄셈, 곱셈, 몫을 각각 출력하세요. 나머지는 `$"{a} % {b} = {a % b}"` 형식으로 출력하세요.

**실행 결과**

```
8
2
15
1
5 % 3 = 2
```

---

## 10. 문자열 연결 연산자

### 10-1. 문자열 연결

`"Hello"` + `"World"`와 `"Hi"` + `" "` + `"everyone"`을 각각 출력하세요.

**실행 결과**

```
HelloWorld
Hi everyone
```

### 10-2. 문자열과 다른 타입의 연결

`"123" + "456"`, `"123" + 456`, `123 + "456"`, `123 + 456`, `"123" + true`를 각각 `Console.WriteLine`으로 출력하세요.

**실행 결과**

```
123456
123456
123456
579
123True
```

---

## 11. 연산자 우선순위

`1 + 2 * 3`의 결과를 변수 `result`에 저장하여 출력하세요. 그 다음 `(1 + 2) * 3`의 결과를 `result`에 저장하여 출력하세요.

**실행 결과**

```
7
9
```
