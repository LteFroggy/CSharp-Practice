# 코딩 연습 - 재귀 함수 ~ 종합 예제

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 재귀 함수

### 1-1. 재귀 함수란?

`CountDown(int n)` 함수를 정의하세요. n부터 1까지 카운트다운한 후 "발사!"를 출력합니다. 종료 조건은 n이 0 이하일 때입니다.

**실행 결과**

```
5
4
3
2
1
발사!
```

### 1-2. 팩토리얼 계산

`Factorial(int n)` 재귀 함수를 정의하여 팩토리얼을 계산하세요. 종료 조건: n이 1 이하이면 1 반환.

**실행 결과**

```
5! = 120
4! = 24
3! = 6
```

### 1-3. 반복문 vs 재귀 함수

`FactorialLoop(int n)`과 `FactorialRecursive(int n)` 두 가지 버전으로 팩토리얼을 구현하고 결과를 비교하세요.

**실행 결과**

```
반복문: 120
재귀: 120
```

## 2. 변수의 범위 (스코프)

### 2-1. 지역 변수

`ShowMessage()` 함수 안에서 지역 변수 `message`를 선언하고 출력하세요. 함수 밖에서는 접근할 수 없음을 확인하세요.

**실행 결과**

```
함수 안의 지역 변수
```

### 2-2. 블록 범위

`TestScope()` 함수에서 변수 `x`를 선언하고, if 블록 안에서 변수 `y`를 선언하여 블록 범위를 확인하세요.

**실행 결과**

```
x = 10, y = 20
x = 10
```

### 2-3. 매개변수의 범위

`ProcessValue(int value)` 함수를 정의하여 값 전달(Pass by Value) 동작을 확인하세요. 함수 안에서 매개변수를 변경해도 원본은 변경되지 않습니다.

**실행 결과**

```
호출 전: 10
함수 안: 20
호출 후: 10
```

## 3. 식 본문 메서드

### 3-1. 식 본문 메서드란?

일반 함수 `Add(int a, int b)`와 식 본문 메서드 `AddShort(int a, int b)`를 각각 정의하여 같은 결과를 출력하세요.

**실행 결과**

```
30
30
```

### 3-2. 반환값이 없는 식 본문 메서드

`SayHello()`와 `SayHelloTo(string name)`를 식 본문 메서드로 정의하세요.

**실행 결과**

```
안녕하세요!
홍길동님, 안녕하세요!
```

### 3-3. 다양한 식 본문 메서드

`Square(int x)`, `IsPositive(int x)`, `GetGreeting(string name)` 함수를 식 본문 메서드로 정의하세요.

**실행 결과**

```
25
False
```
