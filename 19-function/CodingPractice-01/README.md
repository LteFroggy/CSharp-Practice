# 코딩 연습 - 함수 기초 ~ 명명된 매개변수

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 함수란?

### 1-1. 함수 없이 반복 코드

`Console.WriteLine("안녕하세요.");`를 3번 반복 작성하여 출력하세요.

**실행 결과**

```
안녕하세요.
안녕하세요.
안녕하세요.
```

### 1-2. 함수를 사용한 코드

`SayHello()` 로컬 함수를 정의하고 3번 호출하여 같은 결과를 출력하세요.

**실행 결과**

```
안녕하세요.
안녕하세요.
안녕하세요.
```

## 3. 매개변수가 있는 함수

### 3-1. 매개변수란?

`ShowMessage(string message)` 함수를 정의하고, "안녕하세요!", "반갑습니다!", "좋은 하루 되세요!"를 각각 전달하여 출력하세요.

**실행 결과**

```

반갑습니다!
좋은 하루 되세요!

**실행 결과**

```
10 + 20 = 30
5 + 3 = 8
100 + 200 = 300
```

### 3-3. 다양한 타입의 매개변수

```

## 4. 반환값이 있는 함수

### 4-1. 반환값이란?

`GetGreeting()` 함수를 정의하여 "안녕하세요!" 문자열을 반환하고, 반환값을 변수에 저장하여 출력하세요.

**실행 결과**

```

```

### 4-2. 계산 결과 반환

**실행 결과**

```
결과: 30
3 + 5 = 8
```

### 4-3. 제곱 함수

**실행 결과**

```
4
25
100
```

### 4-4. 조건에 따른 반환 (if/else)

```

```
