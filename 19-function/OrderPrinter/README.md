# 주문서 출력기

커피 주문서를 출력하는 프로그램을 작성하세요.

**요구사항**

`PrintOrder` 함수를 작성하세요:

- 매개변수:
  - `menu` (string) - 메뉴 이름 (필수)
  - `size` (string) - 사이즈, 기본값 "Regular"
  - `shots` (int) - 샷 수, 기본값 1
  - `takeout` (bool) - 포장 여부, 기본값 false

- 동작: 포장/매장 여부와 함께 주문 정보를 한 줄로 출력

다음 5가지 방식으로 주문하세요:

1. 메뉴만 지정: "아메리카노"
2. 메뉴와 사이즈 지정: "카페라떼", "Large"
3. 메뉴와 샷 수 지정 (명명된 매개변수): "바닐라라떼", 샷 2
4. 모든 매개변수 지정: "카푸치노", "Small", 3샷, 포장
5. 포장 여부만 지정 (명명된 매개변수): "에스프레소", 포장

## 예상 실행 결과

```
=== 커피 주문서 ===

[매장] Regular 아메리카노 (샷: 1)
[매장] Large 카페라떼 (샷: 1)
[매장] Regular 바닐라라떼 (샷: 2)
[포장] Small 카푸치노 (샷: 3)
[포장] Regular 에스프레소 (샷: 1)
```
