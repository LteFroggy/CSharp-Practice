# 면적 계산기

다양한 도형의 면적을 계산하는 프로그램을 구현하세요.

**요구사항**

클래스 기반으로 `GetArea` 함수를 오버로드하여 구현하세요:

1. `GetArea(double radius)` - 원의 면적
   - 공식: 3.14159 × 반지름 × 반지름

2. `GetArea(int width, int height)` - 직사각형의 면적 (정수)
   - 공식: 가로 × 세로

3. `GetArea(double width, double height)` - 직사각형의 면적 (실수)
   - 공식: 가로 × 세로

4. `GetArea(double top, double bottom, double height)` - 사다리꼴의 면적
   - 공식: (윗변 + 아랫변) × 높이 / 2

> 힌트: 함수 오버로드는 클래스 내부의 `static` 메서드로 작성해야 함

## 예상 실행 결과

```
=== 면적 계산기 ===

[원]
반지름 5 → 면적: 78.54
반지름 10 → 면적: 314.16

[직사각형 (정수)]
4 x 6 → 면적: 24
7 x 3 → 면적: 21

[직사각형 (실수)]
4.5 x 6.2 → 면적: 27.90
7.1 x 3.3 → 면적: 23.43

[사다리꼴]
윗변 3, 아랫변 5, 높이 4 → 면적: 16.00
윗변 6, 아랫변 10, 높이 5 → 면적: 40.00
```
