# 온도 변환기

섭씨와 화씨 온도를 변환하는 프로그램을 작성하세요.

**요구사항**

- 다음 2개의 함수를 정의하세요:
  - `ToFahrenheit(double celsius)` - 섭씨를 화씨로 변환하여 반환
  - `ToCelsius(double fahrenheit)` - 화씨를 섭씨로 변환하여 반환

- 각 함수는 식 본문 메서드(`=>`) 형태로 작성하세요.

- 변환 공식:
  - 화씨 = 섭씨 × 9.0 / 5.0 + 32.0
  - 섭씨 = (화씨 - 32.0) × 5.0 / 9.0

## 예상 실행 결과

```
=== 온도 변환기 ===

[섭씨 → 화씨]
  0°C = 32.0°F
 25°C = 77.0°F
100°C = 212.0°F

[화씨 → 섭씨]
 32°F = 0.0°C
 77°F = 25.0°C
212°F = 100.0°C
```
