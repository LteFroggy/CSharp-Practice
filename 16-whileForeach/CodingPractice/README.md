# 코딩 연습 - while 문과 do 문, foreach 문으로 반복 처리하기

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. while 문

### 1-1. while 문의 기본 구조

정수형 변수 `count`를 0으로 초기화하고, `while` 문을 사용하여 `count`가 3보다 작은 동안 "안녕하세요."를 출력한 후 `count`를 1씩 증가시키세요.

**실행 결과**

```
안녕하세요.
안녕하세요.
안녕하세요.
```

### 1-2. 카운트다운 출력하기

정수형 변수 `index`를 5로 초기화하고, `while` 문을 사용하여 `index`가 0보다 큰 동안 "카운트: {index}"를 출력하며 `index`를 1씩 감소시키세요. 반복이 끝나면 "발사!"를 출력하세요.

**실행 결과**

```
카운트: 5
카운트: 4
카운트: 3
카운트: 2
카운트: 1
발사!
```

### 1-3. 1부터 100까지의 합 구하기

상수 `k_Max`를 100으로 선언하고, `while` 문을 사용하여 1부터 `k_Max`까지의 합을 구한 후 결과를 출력하세요.

**실행 결과**

```
1부터 100까지의 합: 5050
```

### 1-4. 짝수의 합 구하기

`while` 문과 `if` 문을 사용하여 1부터 100까지의 짝수의 합을 구하고 출력하세요.

**실행 결과**

```
1부터 100까지 짝수의 합: 2550
```

### 1-5. 7의 배수의 합 구하기

`while` 문과 `if` 문을 사용하여 1부터 50까지 7의 배수의 합을 구하고 출력하세요.

**실행 결과**

```
1부터 50까지 7의 배수의 합: 196
```

### 1-6. 피보나치 수열 출력하기

피보나치 수열(앞의 두 수를 더해 다음 수를 만드는 수열)에서 20 이하인 값들을 `while` 문을 사용하여 출력하세요. 첫 번째 값 `first`를 0, 두 번째 값 `second`를 1로 초기화하세요.

**실행 결과**

```
피보나치 수열 (20 이하):
1
1
2
3
5
8
13
```

## 2. do-while 문

### 2-1. do-while 문의 기본 구조

정수형 변수 `count`를 0으로 초기화하고, `do-while` 문을 사용하여 "안녕하세요."를 출력한 후 `count`를 1씩 증가시키세요. `count`가 3보다 작은 동안 반복합니다.

**실행 결과**

```
안녕하세요.
안녕하세요.
안녕하세요.
```

### 2-2. while 문과 do-while 문의 차이

정수형 변수 `x`를 10으로 초기화하고, `while` 문으로 `x < 5` 조건을 검사하여 반복을 시도한 뒤 "while 문 종료"를 출력하세요. 다시 `x`를 10으로 초기화하고, `do-while` 문으로 같은 조건(`x < 5`)을 검사하여 반복을 시도한 뒤 "do-while 문 종료"를 출력하세요. 두 반복문의 차이를 확인하세요.

**실행 결과**

```
while 문:
while 문 종료

do-while 문:
x = 10
do-while 문 종료
```

### 2-3. 1부터 5까지의 합 구하기 (do-while)

`do-while` 문을 사용하여 1부터 5까지의 합을 구하고 "합계: {sum}"을 출력하세요.

**실행 결과**

```
합계: 15
```

### 2-4. 3과 4의 공배수 합 구하기

`do-while` 문과 `if` 문을 사용하여 1부터 100까지 3과 4의 공배수(12의 배수)의 합을 구하고 출력하세요.

**실행 결과**

```
1부터 100까지 3과 4의 공배수 합: 432
```

## 3. foreach 문

### 3-1. foreach 문의 기본 구조

문자열 배열 `languages`에 "C#", "Java", "Python"을 저장하고, `foreach` 문을 사용하여 각 요소를 출력하세요.

**실행 결과**

```
C#
Java
Python
```

### 3-2. 문자열에서 문자 하나씩 출력하기

문자열 변수 `text`에 "Hello"를 저장하고, `foreach` 문을 사용하여 각 문자를 공백으로 구분하여 출력하세요.

**실행 결과**

```
문자 하나씩 출력:
H e l l o
```

### 3-3. 문자열에서 문자 분석하기

문자열 변수 `code`에 "ABC123"을 저장하고, `foreach` 문을 사용하여 각 문자가 문자인지 숫자인지 분석하여 출력하세요. `char.IsLetter()`와 `char.IsDigit()` 메서드를 활용하세요.

**실행 결과**

```
코드 분석:
A - 문자
B - 문자
C - 문자
1 - 숫자
2 - 숫자
3 - 숫자
```

### 3-4. 정수 배열의 합 구하기

정수 배열 `numbers`에 10, 20, 30, 40, 50을 저장하고, `foreach` 문을 사용하여 배열 요소의 합을 구한 후 출력하세요.

**실행 결과**

```
배열 요소의 합: 150
```

### 3-5. 배열에서 최댓값 찾기

정수 배열 `scores`에 85, 92, 78, 95, 88을 저장하고, `foreach` 문을 사용하여 최고 점수를 찾아 출력하세요.

**실행 결과**

```
최고 점수: 95
```

### 3-6. var 키워드와 foreach

`var` 키워드를 사용하여 문자열 배열 `items`에 "사과", "바나나", "체리"를 저장하고, `foreach` 문에서 `var`로 요소를 순회하며 출력하세요.

**실행 결과**

```
사과
바나나
체리
```

## 4. 반복문 비교

### 4-1. for 문 vs while 문

`for` 문으로 1부터 5까지 공백으로 구분하여 출력한 후, `while` 문으로 같은 결과를 출력하세요. 두 반복문의 차이를 비교해 보세요.

**실행 결과**

```
for 문:
1 2 3 4 5
while 문:
1 2 3 4 5
```

### 4-2. for 문 vs foreach 문

문자열 배열 `fruits`에 "사과", "바나나", "체리"를 저장하고, `for` 문으로 인덱스와 함께 출력한 후, `foreach` 문으로 값만 출력하세요.

**실행 결과**

```
for 문:
1. 사과
2. 바나나
3. 체리

foreach 문:
사과
바나나
체리
```
