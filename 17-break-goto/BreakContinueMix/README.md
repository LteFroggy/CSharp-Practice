# BreakContinueMix

1부터 시작하여 정수를 순회하면서 다음 조건에 따라 처리하는 프로그램을 작성하세요:

1. **4의 배수**는 건너뜀 (continue)
2. **4의 배수가 아닌 숫자**는 합계에 더함
3. **합계가 200을 초과**하면 반복 종료 (break)

최종적으로 마지막 숫자, 건너뛴 횟수, 합계를 출력합니다.

**요구사항**

- 4의 배수를 만나면 continue로 건너뜀
- 합계가 200을 초과하면 break로 종료
- 건너뛴 4의 배수 개수도 카운트
- 결과 출력

## 예상 실행 결과

```
=== 결과 ===
마지막 숫자: 23
건너뛴 횟수: 5
최종 합계: 216
```
