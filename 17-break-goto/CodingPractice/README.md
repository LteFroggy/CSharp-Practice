# 코딩 연습 - break, continue, goto로 반복문 제어하기

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> :bulb: 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. break 문

### 1-1. 기본 사용법

`for` 문으로 1부터 10까지 반복하면서 `i`를 출력하되, `i`가 3이면 `break`로 반복문을 종료하세요. 반복문이 끝난 후 `"반복문이 종료됨"`을 출력하세요.

**실행 결과**

```
1
2
3
반복문이 종료됨
```

### 1-2. while 문에서 break 사용

`while (true)` 무한 루프에서 `number` 변수를 0부터 시작하여 1씩 증가시키며 출력하고, `number`가 5 이상이면 `break`로 루프를 탈출하세요. 탈출 후 `"무한 루프를 탈출함"`을 출력하세요.

**실행 결과**

```
1
2
3
4
5
무한 루프를 탈출함
```

### 1-3. 조건을 만족할 때까지 합계 구하기

1부터 10까지의 합을 구하되, 합계가 22 이상이 되면 `break`로 반복을 멈추세요. 반복이 끝난 후 몇까지 더했는지와 합계, 목표치 달성 메시지를 출력하세요.

**실행 결과**

```
1부터 7까지의 합: 28
목표치 22 이상 달성
```

### 1-4. 중첩 반복문에서 break

외부 `for` 문은 `i`를 1부터 3까지, 내부 `for` 문은 `j`를 1부터 5까지 반복하세요. 내부 반복문에서 `j`가 3이면 `break`로 내부 반복문만 종료하세요. 외부 반복은 `"외부 반복 {i}"`, 내부 반복은 `"  내부 반복 {j}"`를 출력하세요.

**실행 결과**

```
외부 반복 1
  내부 반복 1
  내부 반복 2
외부 반복 2
  내부 반복 1
  내부 반복 2
외부 반복 3
  내부 반복 1
  내부 반복 2
```

## 2. continue 문

### 2-1. 기본 사용법

`for` 문으로 1부터 5까지 반복하면서, 짝수이면 `continue`로 건너뛰고 홀수만 출력하세요.

**실행 결과**

```
1
3
5
```

### 2-2. 3의 배수를 제외한 합계

1부터 100까지의 정수 중 3의 배수를 `continue`로 건너뛰고, 나머지 숫자의 합계를 구하여 출력하세요.

**실행 결과**

```
1~100 중 3의 배수 제외 합계: 3367
```

### 2-3. 7의 배수를 제외한 합계

1부터 100까지의 정수 중 7의 배수를 `continue`로 건너뛰고, 나머지 숫자의 합계를 구하여 출력하세요.

**실행 결과**

```
1~100 중 7의 배수 제외 합계: 4315
```

### 2-4. while 문에서 continue 사용

`while` 문으로 `i`를 0부터 시작하여 10 미만일 때 반복하면서, `i`를 1씩 증가시키고 짝수이면 `continue`로 건너뛴 후 홀수만 `Console.Write`로 공백 구분하여 한 줄에 출력하세요. 반복이 끝나면 `Console.WriteLine()`으로 줄바꿈하세요.

**실행 결과**

```
1 3 5 7 9
```

## 3. break와 continue 비교

`break`와 `continue`의 차이를 비교하는 코드를 작성하세요. 먼저 `"=== break 예제 ==="`를 출력하고, `for` 문으로 1부터 5까지 반복하면서 `i`가 3이면 `break`하고 그 외에는 `i`를 출력하세요. 그 다음 빈 줄과 함께 `"=== continue 예제 ==="`를 출력하고, 같은 `for` 문에서 `i`가 3이면 `continue`하고 그 외에는 `i`를 출력하세요.

**실행 결과**

```
=== break 예제 ===
1
2

=== continue 예제 ===
1
2
4
5
```

## 4. goto 문

### 4-1. 기본 사용법

`goto` 문을 사용하여 1부터 5까지 출력하세요. `i` 변수를 1로 초기화하고 `StartLoop:` 레이블에서 `i`를 출력한 후 1 증가시키며, `i`가 5 이하이면 `goto StartLoop`으로 이동하세요. 반복이 끝나면 `"종료"`를 출력하세요.

**실행 결과**

```
1
2
3
4
5
종료
```

### 4-2. 여러 레이블 사용

`"시작"`을 출력한 후 사용자로부터 1 또는 2를 입력받아, 1이면 `Option1:` 레이블로, 2이면 `Option2:` 레이블로, 그 외에는 `End:` 레이블로 `goto` 이동하세요. 각 옵션에서는 선택한 옵션을 출력하고 `End:` 레이블로 이동하며, `End:`에서는 `"프로그램 종료"`를 출력하세요.

**실행 결과 (1 입력 시)**

```
시작
1 또는 2를 입력하세요: 1
옵션 1을 선택했습니다.
프로그램 종료
```

### 4-3. switch 문에서 goto case

정수 변수 `card`에 11을 저장하고 `switch` 문으로 분기하세요. `case 13`은 `"King"`, `case 12`는 `"Queen"`, `case 11`은 `"Jack"`을 출력하고 `goto case 12`로 이동, `case -1`은 `goto default`로 이동, `default`는 `"일반 카드"`를 출력하세요.

**실행 결과**

```
Jack
Queen
```

### 4-4. goto 대신 플래그 변수 사용

`bool shouldExit` 플래그 변수를 사용하여 중첩 반복문에서 특정 조건에 모든 반복을 종료하세요. 외부 `for` 문은 `i`를 0부터 2까지, 내부 `for` 문은 `j`를 0부터 2까지 반복하면서 `"i={i}, j={j}"`를 출력하되, `i == 1 && j == 1`이면 `shouldExit`를 `true`로 설정하여 모든 반복을 종료하세요.

**실행 결과**

```
i=0, j=0
i=0, j=1
i=0, j=2
i=1, j=0
```

## 5. 실전 예제

### 5-1. 사용자 입력 유효성 검사

`while (true)` 무한 루프에서 사용자에게 1~10 사이의 숫자를 입력받으세요. 숫자가 아닌 입력이면 `"숫자를 입력해주세요."`를 출력하고 `continue`, 범위를 벗어나면 `"범위를 벗어났습니다."`를 출력하고 `continue`, 유효한 입력이면 `break`로 루프를 종료하세요. 최종적으로 입력한 숫자를 출력하세요.

**실행 결과**

```
1~10 사이의 숫자를 입력하세요: abc
숫자를 입력해주세요.
1~10 사이의 숫자를 입력하세요: 15
범위를 벗어났습니다.
1~10 사이의 숫자를 입력하세요: 7
입력한 숫자: 7
```

### 5-2. 특정 값 찾기

정수 배열 `{ 5, 12, 8, 23, 15, 42, 7 }`에서 값 23을 검색하세요. `for` 문으로 배열을 순회하면서 목표 값을 찾으면 인덱스를 저장하고 `break`로 즉시 종료하세요. 찾은 경우와 못 찾은 경우를 구분하여 결과를 출력하세요.

**실행 결과**

```
23을(를) 인덱스 3에서 찾음
```

### 5-3. 양수만 합계 구하기

정수 배열 `{ 3, -2, 7, -5, 4, -1, 9 }`에서 `foreach` 문으로 순회하면서 음수이면 `continue`로 건너뛰고, 양수만 합계에 더한 후 결과를 출력하세요.

**실행 결과**

```
양수의 합계: 23
```
