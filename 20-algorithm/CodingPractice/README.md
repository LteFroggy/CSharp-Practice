# 코딩 연습 - 알고리즘과 절차 지향 프로그래밍

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 합계 알고리즘

### 1-1. 기본 합계

정수 배열 `{ 10, 20, 30, 40, 50 }`의 모든 요소의 합계를 구하여 출력하세요.

**실행 결과**

```
합계: 150
```

### 1-2. 조건부 합계

점수 배열 `{ 100, 75, 50, 37, 90, 95 }`에서 80점 이상인 점수만 합산하여 출력하세요.

**실행 결과**

```
6명의 점수 중 80점 이상의 총점: 285
```

---

## 2. 개수 알고리즘

배열 `{ 10, 9, 4, 7, 6, 5 }`에서 짝수의 개수를 세어 출력하세요.

**실행 결과**

```
짝수의 개수: 3
```

---

## 3. 평균 알고리즘

### 3-1. 전체 평균

배열 `{ 50, 65, 78, 90, 95 }`의 전체 평균을 구하여 출력하세요. 평균은 `double` 타입으로 계산하세요.

**실행 결과**

```
평균: 75.6
```

### 3-2. 조건부 평균

배열 `{ 50, 65, 78, 90, 95 }`에서 80점 이상 95점 이하인 점수의 평균을 구하여 출력하세요.

**실행 결과**

```
80점 이상 95점 이하인 점수의 평균: 92.5
```

---

## 4. 최댓값 알고리즘

배열 `{ -2, -5, -3, -7, -1 }`에서 최댓값을 찾아 출력하세요. `int.MinValue`로 초기화하세요.

**실행 결과**

```
최댓값: -1
```

---

## 5. 최솟값 알고리즘

배열 `{ -2, -5, -3, -7, -1 }`에서 최솟값을 찾아 출력하세요. `int.MaxValue`로 초기화하세요.

**실행 결과**

```
최솟값: -7
```

---

## 6. 근삿값 알고리즘

배열 `{ 10, 20, 30, 27, 17 }`에서 목표 값 `25`에 가장 가까운 값을 찾아 출력하세요. 각 값과 목표 값의 차이의 절댓값(`Math.Abs`)을 비교하세요.

**실행 결과**

```
25에 가장 가까운 값: 27
```

---

## 7. 순위 알고리즘

점수 배열 `{ 90, 87, 100, 95, 80 }`의 각 점수에 대한 순위를 계산하여 출력하세요. 모든 순위를 1등으로 초기화한 후, 자신보다 높은 점수가 있으면 순위를 1씩 증가시키세요.

**실행 결과**

```
 90점: 3등
 87점: 4등
100점: 1등
 95점: 2등
 80점: 5등
```

---

## 8. 정렬 알고리즘

배열 `{ 7, 5, 6, 1, 10 }`을 선택 정렬로 오름차순 정렬한 후 결과를 출력하세요.

**실행 결과**

```
정렬 결과: 1 5 6 7 10
```

---

## 9. 검색 알고리즘

### 9-1. 순차 검색

배열 `{ 3, 7, 1, 9, 5 }`에서 값 `9`를 처음부터 순서대로 검색하여 찾은 인덱스를 출력하세요.

**실행 결과**

```
9을(를) 3번 인덱스에서 찾음
```

### 9-2. 이진 검색

정렬된 배열 `{ 1, 3, 5, 7, 9 }`에서 값 `7`을 이진 검색으로 찾아 인덱스를 출력하세요. `low`, `high`, `mid` 변수를 사용하여 탐색 범위를 반씩 줄이세요.

**실행 결과**

```
7을(를) 3번 인덱스에서 찾음
```

---

## 10. 병합 알고리즘

두 정렬된 배열 `{ 1, 3, 5 }`와 `{ 2, 4 }`를 하나의 정렬된 배열로 병합하여 출력하세요.

**실행 결과**

```
병합 결과: 1 2 3 4 5
```

---

## 11. IPO 구조

점수 배열 `{ 85, 90, 78, 92, 88 }`에 대해 IPO(Input-Process-Output) 구조로 성적 처리 프로그램을 작성하세요. 합계, 평균(소수점 첫째 자리), 최고점, 최저점을 출력하세요.

**실행 결과**

```
합계: 433
평균: 86.6
최고점: 92
최저점: 78
```
