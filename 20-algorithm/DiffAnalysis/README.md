# 인접 차이 분석

정수 배열에서 인접한 요소 간의 차이를 계산하고 분석하는 프로그램을 작성하시오.

**요구사항**

1. 인접한 두 요소의 차이(뒤 - 앞)를 모두 계산
2. 각 변화량을 부호(+/-)와 함께 출력
3. 최대 증가량과 최대 감소량을 찾기
4. 증가/감소 횟수를 계산

**입력 데이터**

```
int[] data = { 5, 8, 3, 11, 7, 15, 10 };
```

## 예상 실행 결과

```
===== 인접 차이 분석 =====

데이터: 5 8 3 11 7 15 10

[인접 변화량]
5 → 8: +3
8 → 3: -5
3 → 11: +8
11 → 7: -4
7 → 15: +8
15 → 10: -5

[분석 결과]
최대 증가: +8
최대 감소: -5
증가 횟수: 3번
감소 횟수: 3번
```
