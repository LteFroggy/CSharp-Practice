# 빈도수 계산기

정수 배열에서 각 고유 값의 출현 횟수를 계산하는 프로그램을 작성하시오.

**요구사항**

- 이미 출력한 값은 다시 출력하지 않음 (중복 제거)
- 각 고유 값이 배열에 몇 번 나타나는지 계산
- 등장 순서대로 결과를 출력

**입력 데이터**

```
int[] data = { 3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5 };
```

## 예상 실행 결과

```
===== 빈도수 계산기 =====

배열: 3 1 4 1 5 9 2 6 5 3 5

값  | 빈도
----|-----
 3  |  2
 1  |  2
 4  |  1
 5  |  3
 9  |  1
 2  |  1
 6  |  1
```
