# 버블 정렬과 검색

정수 배열이 주어졌을 때, 다음 기능을 순서대로 수행하는 프로그램을 작성하시오.

1. 원본 배열 출력
2. 버블 정렬로 오름차순 정렬 (교환 횟수 카운트)
3. 정렬된 배열 출력
4. 순차 검색으로 존재하는 값과 존재하지 않는 값 찾기
5. 이진 검색으로 존재하는 값과 존재하지 않는 값 찾기

**입력 데이터**

```
int[] data = { 38, 27, 43, 3, 9, 82, 10 };
```

## 예상 실행 결과

```
===== 버블 정렬과 검색 =====

[1] 원본 배열
38 27 43 3 9 82 10

[2] 버블 정렬 (교환 11회)
3 9 10 27 38 43 82

[3] 순차 검색
  43 검색: 5번 인덱스에서 찾음
  50 검색: 찾지 못함

[4] 이진 검색
  43 검색: 5번 인덱스에서 찾음
  50 검색: 찾지 못함
```
