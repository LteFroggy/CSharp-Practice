# 문자열 정렬 프로그램

사용자로부터 쉼표(,)로 구분된 여러 개의 이름을 입력받아 다양한 방식으로 정렬하여 출력하는 프로그램을 작성하세요.

**프로그램 요구사항:**

1. 사용자로부터 쉼표로 구분된 이름 목록을 입력받음
2. 입력받은 이름의 앞뒤 공백을 제거함
3. 다음 두 가지 정렬 결과를 출력함:
   - 사전 순 정렬 (오름차순)
   - 사전 순 역정렬 (내림차순)

**정렬 방법:**

- `Array.Sort()`를 사용하여 사전 순 정렬
- `Array.Reverse()`를 사용하여 역정렬
- 원본 배열을 보존하기 위해 `Array.Copy()`로 배열을 복사한 뒤 정렬할 것

## 예상 실행 결과

```
이름 목록을 쉼표로 구분하여 입력하세요:
홍길동, 이순신, 김유신, 강감찬, 을지문덕

[입력된 이름 목록]
홍길동, 이순신, 김유신, 강감찬, 을지문덕

[사전 순 정렬]
강감찬, 김유신, 을지문덕, 이순신, 홍길동

[사전 순 역정렬]
홍길동, 이순신, 을지문덕, 김유신, 강감찬
```
