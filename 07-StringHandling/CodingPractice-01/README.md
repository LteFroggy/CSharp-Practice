# 코딩 연습 - 문자열 다루기 (1)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

## 1. string 클래스 기본

### 1-1. string과 String의 관계

`String` 클래스와 `string` 키워드를 각각 사용하여 문자열 변수를 선언하고, 두 변수를 문자열 보간법으로 함께 출력하세요.

**실행 결과**

```
안녕하세요. 반갑습니다.
```

### 1-2. 문자열의 불변성

문자열 변수 `original`에 `"Hello"`를 저장하고, `ToUpper()` 메서드로 대문자 변환한 결과를 `modified`에 저장한 후, 원본과 수정본을 각각 출력하여 원본이 변경되지 않음을 확인하세요.

**실행 결과**

```
원본: Hello
수정: HELLO
```

## 2. 문자열 생성 방법

### 2-1. 문자 배열에서 문자열 생성

문자 배열 `{ 'A', 'B', 'C' }`를 만들고, `new string(charArray)`를 사용하여 문자열로 변환한 후 출력하세요.

**실행 결과**

```
ABC
```

### 2-2. 빈 문자열 초기화

빈 문자열 리터럴 `""`과 `string.Empty`를 각각 변수에 저장하고, 두 변수의 `Length` 속성을 출력하세요.

**실행 결과**

```
빈 문자열 길이: 0
Empty 길이: 0
```

## 3. 문자열 속성과 인덱싱

### 3-1. Length 속성

영문 문자열 `"Hello."`과 한글 문자열 `"안녕하세요."`를 각각 변수에 저장하고, `Length` 속성으로 길이를 출력하세요.

**실행 결과**

```
s1 길이: 6
s2 길이: 6
```

### 3-2. 인덱서를 통한 문자 접근

문자열 `"Hello"`의 첫 번째 문자와 마지막 문자를 인덱서로 출력하고, `for` 반복문으로 모든 문자를 공백으로 구분하여 출력하세요.

**실행 결과**

```
첫 번째 문자: H
마지막 문자: o
H e l l o
```

## 4. 문자열 연결

### 4-1. 덧셈 연산자 사용

`"길동"`과 `"홍"`을 `+` 연산자로 연결하여 전체 이름을 만들고 출력하세요.

**실행 결과**

```
이름: 홍길동
```

### 4-2. String.Concat() 메서드

`string.Concat()` 메서드를 사용하여 `"안녕"`과 `"하세요."`를 연결하고, `"a"`, `"b"`, `"c"`, `"d"`를 연결하여 각각 출력하세요.

**실행 결과**

```
안녕하세요.
abcd
```

### 4-3. String.Join() 메서드

문자열 배열 `{ "사과", "바나나", "오렌지" }`를 `string.Join()` 메서드로 `", "` 구분자를 사용하여 연결하고 출력하세요.

**실행 결과**

```
사과, 바나나, 오렌지
```

## 5. 문자열 보간법

### 5-1. 기본 보간법

문자열 보간법 `$""`을 사용하여 성과 이름을 조합한 문자열을 출력하고, 두 정수의 덧셈 결과를 보간법 내 표현식으로 출력하세요.

**실행 결과**

```
이름: 홍길동
10 + 20 = 30
```

### 5-2. 서식 지정

문자열 보간법에서 콜론(`:`) 뒤에 서식 문자열을 사용하여 정수를 세 자리 구분, 실수를 통화 형식, `DateTime.Now`를 `yyyy-MM-dd` 형식, 정수 `255`를 16진수로 출력하세요.

**실행 결과**

```
세 자리 구분: 1,234,567
통화 형식: ₩9,999.99
날짜 형식: 2024-12-25
16진수: FF
```

### 5-3. String.Format() 메서드

`string.Format()`을 사용하여 자리 표시자 `{0}`, `{1}`로 이름을 조합하고, `{0:C}` 서식으로 통화 형식, `{0:#,###}` 서식으로 천 단위 구분을 출력하세요.

**실행 결과**

```
이름: 홍길동
₩1,000
1,000,000
```

## 8. 문자열 검색

### 8-1. IndexOf()와 LastIndexOf()

문자열 `"Hello, World!"`에서 `IndexOf('o')`, `LastIndexOf('o')`, `IndexOf("World")`를 각각 호출하여 위치를 출력하세요.

**실행 결과**

```
첫 번째 'o' 위치: 4
마지막 'o' 위치: 8
'World' 위치: 7
```

### 8-2. Contains()

문자열 `"Hello, World!"`에서 `Contains("World")`와 `Contains("world")`의 결과를 각각 출력하세요.

**실행 결과**

```
'World' 포함: True
'world' 포함: False
```

### 8-3. StartsWith()와 EndsWith()

문자열 `"https://www.example.com"`에서 `StartsWith("https")`와 `EndsWith(".com")`의 결과를 각각 출력하세요.

**실행 결과**

```
https로 시작: True
.com으로 끝남: True
```

## 9. 문자열 비교

### 9-1. == 연산자

문자열 `"Hello"` 두 개와 `"hello"` 하나를 선언하고, `==` 연산자로 비교한 결과를 출력하세요.

**실행 결과**

```
s1 == s2: True
s1 == s3: False
```

### 9-2. Equals() 메서드 - 대소문자 무시

문자열 `"Gilbut"`과 `"gilbut"`을 `==`로 비교하여 결과를 출력하고, `Equals()`에 `StringComparison.OrdinalIgnoreCase` 옵션을 사용하여 대소문자 무시 비교 결과를 출력하세요.

**실행 결과**

```
다르다.
대소문자 무시하면 같다.
```

### 9-3. String.Compare() 메서드

`string.Compare()`를 사용하여 `"A"`와 `"C"`, `"C"`와 `"A"`, `"A"`와 `"A"`를 각각 비교한 결과를 출력하세요.

**실행 결과**

```
A와 C 비교: -1
C와 A 비교: 1
A와 A 비교: 0
```
