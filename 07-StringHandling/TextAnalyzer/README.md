# 텍스트 분석기

사용자로부터 문장을 입력받아 다양한 정보를 분석하여 출력하는 텍스트 분석기 프로그램을 작성하세요.

**분석할 정보:**

1. 전체 문자 수 (공백 포함)
2. 전체 문자 수 (공백 제외)
3. 단어 개수
4. 대문자 개수
5. 소문자 개수
6. 숫자 개수
7. 특정 문자가 포함되어 있는지 검색

**요구사항:**

- 모든 분석은 입력된 원본 문자열을 기준으로 수행
- 특정 문자 검색 기능은 사용자가 검색할 문자를 추가로 입력받음
- 분석 결과를 보기 좋게 정리하여 출력

## 예상 실행 결과

```
문장을 입력하세요:
Hello World 2024! Welcome to C# Programming.

=== 텍스트 분석 결과 ===
전체 문자 수 (공백 포함): 45
전체 문자 수 (공백 제외): 39
단어 개수: 7
대문자 개수: 5
소문자 개수: 26
숫자 개수: 4

검색할 문자를 입력하세요:
o
'o' 문자가 포함되어 있습니다. (첫 번째 위치: 4)
```
