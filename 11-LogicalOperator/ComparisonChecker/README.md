# 숫자 비교기

사용자로부터 두 개의 정수를 입력받아 다양한 관계형 연산자로 비교한 결과를 출력하는 프로그램을 작성하세요.

**요구사항**

- 첫 번째 정수와 두 번째 정수를 순서대로 입력받음
- 6가지 관계형 연산자(`==`, `!=`, `>`, `>=`, `<`, `<=`)로 비교한 결과를 출력함
- 추가로 두 수가 모두 양수인지 확인하는 결과도 출력함

## 예상 실행 결과

```
첫 번째 정수를 입력하세요: 5
두 번째 정수를 입력하세요: 3

=== 비교 결과 ===
5 == 3 : False
5 != 3 : True
5 > 3 : True
5 >= 3 : True
5 < 3 : False
5 <= 3 : False
둘 다 양수인가? True
```
