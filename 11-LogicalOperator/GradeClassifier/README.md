# 성적 등급 판정

점수를 입력받아 해당 점수가 각 등급 범위에 해당하는지 판정하는 프로그램을 작성하세요.

**등급 기준**

| 등급 | 점수 범위 |
|:---:|:---:|
| A | 90 ~ 100 |
| B | 80 ~ 89 |
| C | 70 ~ 79 |
| D | 60 ~ 69 |
| F | 0 ~ 59 |

**요구사항**

- 사용자로부터 점수(0~100)를 입력받음
- 각 등급에 해당하는지 여부를 `bool` 값으로 계산하여 출력함
- 범위 조건은 `&&` 연산자를 사용하여 표현함

## 예상 실행 결과

**예시 1: 85점**

```
점수를 입력하세요 (0-100): 85

=== 등급 판정 결과 ===
A등급 (90~100)인가? False
B등급 (80~89)인가? True
C등급 (70~79)인가? False
D등급 (60~69)인가? False
F등급 (0~59)인가? False
```
