# 접근 권한 체크

게임 서버의 접근 권한을 확인하는 프로그램을 작성하세요. 다음 조건에 따라 접근 가능 여부를 판단합니다.

**접근 조건**

- **관리자 접근**: 레벨이 100 이상이면 항상 접근 가능
- **일반 사용자 접근**: 레벨이 10 이상이고, 차단되지 않았으며, 인증된 사용자일 때 접근 가능

**요구사항**

- 레벨, 차단 여부, 인증 여부를 변수로 설정함
- 위 조건에 따라 접근 가능 여부를 계산하여 출력함
- 각 조건별 상태도 함께 출력함

## 예상 실행 결과

**케이스 1: 일반 사용자 접근 가능**

```
=== 사용자 정보 ===
레벨: 25
차단 여부: False
인증 여부: True

=== 접근 권한 확인 ===
관리자인가? False
일반 접근 조건 충족? True
최종 접근 가능? True
```

**케이스 2: 관리자 접근**

```
=== 사용자 정보 ===
레벨: 100
차단 여부: True
인증 여부: False

=== 접근 권한 확인 ===
관리자인가? True
일반 접근 조건 충족? False
최종 접근 가능? True
```

**케이스 3: 접근 불가**

```
=== 사용자 정보 ===
레벨: 5
차단 여부: False
인증 여부: True

=== 접근 권한 확인 ===
관리자인가? False
일반 접근 조건 충족? False
최종 접근 가능? False
```
