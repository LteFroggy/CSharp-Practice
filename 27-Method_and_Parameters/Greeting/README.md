# Greeting

사용자 인사 메시지를 생성하는 `CreateGreeting` 메서드를 작성하세요. 다양한 호출 방식을 테스트합니다.

**요구사항**

- `CreateGreeting` 메서드는 이름, 인사말, 시간대를 매개변수로 받음
- 이름은 필수, 인사말과 시간대는 선택적 매개변수로 구현
- 기본 인사말: "안녕하세요"
- 기본 시간대: "아침"
- 반환 형식: "[시간대] [인사말], [이름]님!"

## 예상 실행 결과

```
[아침] 안녕하세요, 홍길동님!
[아침] 반갑습니다, 김철수님!
[저녁] 안녕하세요, 이영희님!
[점심] 식사하셨어요, 박민수님!
```
