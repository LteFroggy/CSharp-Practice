# 메서드와 매개변수 - 코딩 연습

교안의 예제 코드를 직접 작성하고 실행하는 프로젝트입니다.

> **참고**: 과제마다 동일한 이름의 변수나 메서드, 클래스가 사용될 수 있습니다.
> 이전 과제의 코드를 주석 처리하거나 삭제한 후 다음 과제를 작성하세요.

---

## 과제 1: 기본 메서드 선언

`Calculator` 클래스에 두 정수를 더하는 `Add` 인스턴스 메서드를 만들고, 인스턴스를 생성하여 `Add(3, 5)`를 호출한 결과를 출력하세요.

**실행 결과**

```
8
```

---

## 과제 2: 정적 메서드 호출

`MathHelper` 클래스에 정수의 제곱을 반환하는 `Square` 정적 메서드를 만들고, 클래스 이름으로 직접 호출하여 `Square(5)`의 결과를 출력하세요.

**실행 결과**

```
5의 제곱: 25
```

---

## 과제 3: 값 전달 (Pass by Value)

정수 변수 `num`에 10을 저장하고, `ChangeValue` 메서드에 전달하세요. 메서드 내부에서 값을 20으로 변경하고, 호출 전/내부/호출 후의 값을 각각 출력하여 원본이 변경되지 않음을 확인하세요.

**실행 결과**

```
[1] 호출 전: 10
[2] 메서드 내부: 20
[3] 호출 후: 10
```

---

## 과제 4: 값 전달과 참조 전달 비교

정수 변수 `originalValue`에 10을 저장하고, `PassByValue` 메서드와 `PassByReference` 메서드를 순서대로 호출하세요. 각 메서드는 전달받은 값에 10을 더합니다. 호출 전후의 값을 출력하여 두 방식의 차이를 확인하세요.

**실행 결과**

```
PassByValue 실행 전: 10
PassByValue 내부에서 수정된 값: 20
PassByValue 실행 후: 10

PassByReference 실행 전: 10
PassByReference 내부에서 수정된 값: 20
PassByReference 실행 후: 20
```

## 과제 5: ref 키워드

정수 변수 `num`에 10을 저장하고, `ref` 키워드를 사용하여 `ChangeRef` 메서드에 전달하세요. 메서드 내부에서 값을 20으로 변경하고, 호출 전/내부/호출 후의 값을 출력하여 원본이 변경됨을 확인하세요.

**실행 결과**

```
[1] 호출 전: 10
[2] 메서드 내부: 20
[3] 호출 후: 20
```

## 과제 6: ref를 사용한 두 변수 교환

두 정수 변수 `a`(10)와 `b`(20)를 선언하고, `ref`를 사용하는 `Swap` 메서드를 만들어 두 값을 교환하세요. 교환 전후의 값을 출력하세요.

**실행 결과**

```
교환 전: a = 10, b = 20
교환 후: a = 20, b = 10
```

## 과제 7: out 키워드

초기화하지 않은 정수 변수 `result`를 선언하고, `out` 키워드를 사용하여 `Calculate` 메서드에 전달하세요. 메서드 내부에서 1234를 할당하고, 메서드 내부와 호출 후의 값을 출력하세요.

**실행 결과**

```
메서드 내부: 1234
결과: 1234
```

## 과제 8: out을 사용한 여러 값 반환

문자열 `fullName`에 "홍길동"을 저장하고, `SplitName` 메서드를 만들어 `out`으로 성과 이름을 분리하세요. 첫 글자를 성, 나머지를 이름으로 분리합니다.

**실행 결과**

```
성: 홍, 이름: 길동
```

## 과제 9: out 변수 인라인 선언

**실행 결과**


`Point` 구조체(X, Y 필드)를 선언하고, `in` 키워드를 사용하는 `PrintPoint` 메서드를 만들어 좌표를 출력하세요. `Point(10, 20)`을 생성하여 전달합니다.
