# SwapStrings

학생들의 자리를 교환하는 프로그램을 작성하세요.

**요구사항**

- `SwapSeats(ref string seat1, ref string seat2)` 메서드: 두 문자열의 값을 교환
- `CountSwap(ref int count)` 메서드: 교환 횟수를 1 증가시킴
- Main에서 3명의 학생 이름을 변수로 선언하고, 자리 교환을 수행
- 각 교환 후 현재 배치 상태와 최종 교환 횟수를 출력

## 예상 실행 결과

```
=== 초기 배치 ===
1번: 김민수, 2번: 이지영, 3번: 박서준

[교환] 1번 <-> 2번
1번: 이지영, 2번: 김민수, 3번: 박서준

[교환] 2번 <-> 3번
1번: 이지영, 2번: 박서준, 3번: 김민수

=== 최종 결과 ===
1번: 이지영, 2번: 박서준, 3번: 김민수
총 교환 횟수: 2회
```
