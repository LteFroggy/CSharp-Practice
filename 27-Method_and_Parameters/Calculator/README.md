# Calculator

가변 개수의 정수를 받아 합계, 평균, 최댓값, 최솟값을 계산하는 메서드들을 작성하세요.

**요구사항**

- `Sum`: 전달받은 모든 정수의 합계 반환
- `Average`: 전달받은 모든 정수의 평균 반환 (소수점 포함)
- `Max`: 전달받은 정수 중 최댓값 반환
- `Min`: 전달받은 정수 중 최솟값 반환
- 모든 메서드는 `params` 키워드를 사용하여 가변 개수의 인수를 받음

## 예상 실행 결과

```
숫자: 10, 20, 30, 40, 50
합계: 150
평균: 30
최댓값: 50
최솟값: 10
```
