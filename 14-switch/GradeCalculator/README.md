# 학점 계산기

사용자로부터 점수(0~100)를 입력받아 학점을 출력하는 프로그램을 작성하세요.

**학점 기준**

- 90점 이상: A
- 80점 이상: B
- 70점 이상: C
- 60점 이상: D
- 60점 미만: F

프로그램은 다음과 같이 동작해야 합니다:

1. "점수를 입력하세요: " 메시지를 출력함
2. 사용자가 점수를 입력함
3. 점수를 10으로 나눈 값을 switch 문의 조건으로 사용함
4. 해당하는 학점을 출력함

> 💡 **힌트**: 정수 나눗셈을 이용하면 85점은 8, 92점은 9가 됨. 이 값을 switch 문에서 활용하면 됨.

## 예상 실행 결과

**입력: 95**

```
점수를 입력하세요: 95
학점: A
```

**입력: 83**

```
점수를 입력하세요: 83
학점: B
```

**입력: 55**

```
점수를 입력하세요: 55
학점: F
```
