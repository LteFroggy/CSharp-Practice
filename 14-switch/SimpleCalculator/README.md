# 간단한 계산기

두 정수와 연산자를 입력받아 계산 결과를 출력하는 간단한 계산기 프로그램을 작성하세요.

**지원하는 연산자**

- `+` : 덧셈
- `-` : 뺄셈
- `*` : 곱셈
- `/` : 나눗셈 (정수 나눗셈)

프로그램은 다음과 같이 동작해야 합니다:

1. 첫 번째 숫자를 입력받음
2. 연산자를 입력받음 (+, -, *, / 중 하나)
3. 두 번째 숫자를 입력받음
4. 계산 결과를 출력함
5. 지원하지 않는 연산자가 입력되면 "지원하지 않는 연산자입니다."를 출력함

> ⚠️ **주의**: 나눗셈에서 0으로 나누는 경우는 고려하지 않아도 됨

## 예상 실행 결과

**덧셈 예시**

```
첫 번째 숫자: 15
연산자 (+, -, *, /): +
두 번째 숫자: 7
결과: 15 + 7 = 22
```

**곱셈 예시**

```
첫 번째 숫자: 8
연산자 (+, -, *, /): *
두 번째 숫자: 6
결과: 8 * 6 = 48
```

**잘못된 연산자 예시**

```
첫 번째 숫자: 10
연산자 (+, -, *, /): %
두 번째 숫자: 3
지원하지 않는 연산자입니다.
```
