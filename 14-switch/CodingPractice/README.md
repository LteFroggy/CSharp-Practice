# 코딩 연습 - switch 문으로 다양한 조건 처리하기

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. switch 문의 기본 구조

정수형 변수 `x`를 선언하고 2를 저장한 후, switch 문을 사용하여 `x`가 1이면 `"1입니다."`, 2이면 `"2입니다."`를 출력하세요.

**실행 결과**

```
2입니다.
```

## 2. switch 문 활용하기

### 2-1. 정수형 switch 문

`"정수를 입력하세요."`를 출력하고 사용자로부터 정수를 입력받으세요. switch 문을 사용하여 1이면 `"1을 선택했습니다."`, 2이면 `"2을 선택했습니다."`, 3이면 `"3을 선택했습니다."`, 그 외에는 `"그냥 찍으셨군요."`를 출력하세요.

**실행 결과 (입력: 2)**

```
정수를 입력하세요.
2
2을 선택했습니다.
```

**실행 결과 (입력: 5)**

```
정수를 입력하세요.
5
그냥 찍으셨군요.
```

### 2-2. 문자열 switch 문

`"오늘 날씨는 어떤가요? (맑음, 흐림, 비, 눈, ...)"`를 출력하고 사용자로부터 문자열을 입력받으세요. switch 문을 사용하여 `"맑음"`이면 `"오늘 날씨는 맑군요."`, `"흐림"`이면 `"오늘 날씨는 흐리군요."`, `"비"`이면 `"오늘 날씨는 비가 오는군요."`, 그 외에는 `"혹시 눈이 내리나요?"`를 출력하세요.

**실행 결과 (입력: 맑음)**

```
오늘 날씨는 어떤가요? (맑음, 흐림, 비, 눈, ...)
맑음
오늘 날씨는 맑군요.
```

### 2-3. 문자형 switch 문

문자형 변수 `signalLight`에 `'G'`를 저장하고, 문자열 변수 `message`를 빈 문자열로 초기화하세요. switch 문을 사용하여 `'R'`이면 `"적색: 멈추세요."`, `'Y'`이면 `"황색: 주의하세요."`, `'G'`이면 `"녹색: 이동하세요."`, 그 외에는 `"신호등 고장"`을 `message`에 저장한 후 출력하세요.

**실행 결과**

```
녹색: 이동하세요.
```

## 3. 여러 case 묶어서 처리하기

### 3-1. 여러 case 묶기

정수형 변수 `cardNumber`에 12를 저장하세요. switch 문을 사용하여 11, 12, 13이면 `"그림 카드입니다."`, 그 외에는 `"숫자 카드입니다."`를 출력하세요. 여러 case를 연속으로 나열하여 묶어서 처리하세요.

**실행 결과**

```
그림 카드입니다.
```

### 3-2. 메뉴 선택

`"가장 좋아하는 프로그래밍 언어는? "`를 출력한 후, 탭으로 구분된 4개의 메뉴(`1. C`, `2. C++`, `3. C#`, `4. Java`)를 한 줄에 출력하세요. 사용자로부터 정수를 입력받아 switch 문으로 해당 언어 이름을 출력하세요. 1이면 `"C언어 선택"`, 2이면 `"C++ 선택"`, 3이면 `"C# 선택"`, 4이면 `"Java 선택"`, 그 외에는 `"C, C++, C#, Java가 아니군요."`를 출력하세요.

**실행 결과 (입력: 3)**

```
가장 좋아하는 프로그래밍 언어는?
1. C	2. C++	3. C#	4. Java
3
C# 선택
```

### 3-3. 간단한 계산기

정수형 변수 `first`에 10, 문자형 변수 `op`에 `'+'`, 정수형 변수 `second`에 5를 저장하세요. switch 문을 사용하여 `op`가 `'+'`이면 덧셈, `'-'`이면 뺄셈, `'*'`이면 곱셈 결과를 `"first op second = 결과"` 형식으로 출력하세요. 그 외에는 `"+, -, *만 사용하세요."`를 출력하세요.

**실행 결과**

```
10 + 5 = 15
```

## 4. if-else와 switch 문 비교

문자열 변수 `color`에 `"green"`, 문자열 변수 `result`를 선언하세요. switch 문을 사용하여 `"red"`이면 `"빨강"`, `"green"`이면 `"초록"`, `"blue"`이면 `"파랑"`, 그 외에는 `"기타 색상"`을 `result`에 저장한 후, `"color: result"` 형식으로 출력하세요.

**실행 결과**

```
green: 초록
```
