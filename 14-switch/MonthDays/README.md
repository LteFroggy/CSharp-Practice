# 월별 일수 출력

사용자로부터 월(1~12)을 입력받아 해당 월의 일수를 출력하는 프로그램을 작성하세요.

**월별 일수 기준** (윤년은 고려하지 않음)

- 31일: 1월, 3월, 5월, 7월, 8월, 10월, 12월
- 30일: 4월, 6월, 9월, 11월
- 28일: 2월

1~12 범위를 벗어나는 값이 입력되면 "잘못된 월입니다."를 출력하세요.

## 예상 실행 결과

**입력: 7**

```
월을 입력하세요 (1~12): 7
7월은 31일까지 있습니다.
```

**입력: 2**

```
월을 입력하세요 (1~12): 2
2월은 28일까지 있습니다.
```

**입력: 4**

```
월을 입력하세요 (1~12): 4
4월은 30일까지 있습니다.
```

**입력: 15**

```
월을 입력하세요 (1~12): 15
잘못된 월입니다.
```
