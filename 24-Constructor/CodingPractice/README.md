# 생성자 - 코딩 연습

교안의 예제 코드를 직접 작성하고 실행하는 프로젝트입니다.

> **주의**: 각 과제의 코드를 작성한 후 실행하여 결과를 확인합니다.
> 변수 이름이 겹치는 경우 이전 과제의 코드를 주석 처리하거나 삭제하세요.

---

## 과제 1: 생성자 만들기

`Person` 클래스를 만들고 매개변수 없는 생성자를 작성합니다. 생성자에서 `_name` 필드를 "홍길동"으로 초기화하고 "생성자가 호출되었습니다." 메시지를 출력합니다. `GetName()` 메서드로 이름을 반환합니다.

**실행 결과**

```
생성자가 호출되었습니다.
홍길동
```

---

## 과제 2: 기본 생성자

`Dog` 클래스를 생성자 없이 선언합니다. 외부에서 `Name` 필드에 값을 설정하고 출력합니다. 생성자를 정의하지 않아도 기본 생성자가 자동 제공되는 것을 확인합니다.

**실행 결과**

```
해피
```

---

## 과제 3: 매개변수가 있는 생성자

`Dog` 클래스에 이름을 받는 생성자를 작성합니다. `Cry()` 메서드에서 이름과 함께 "멍멍멍" 메시지를 반환합니다. 두 마리의 개 객체를 만들어 테스트합니다.

**실행 결과**

```
해피이(가) 멍멍멍
워리이(가) 멍멍멍
```

---

## 과제 4: this 키워드로 필드와 매개변수 구분

`Person` 클래스에 `name`과 `age` 필드를 선언합니다. 생성자에서 매개변수와 필드 이름이 같을 때 `this` 키워드로 구분하여 초기화합니다. `PrintInfo()` 메서드로 정보를 출력합니다.

**실행 결과**

```
이름: 홍길동, 나이: 21
```

---

## 과제 5: 원의 면적 계산

`Circle` 클래스에 반지름을 받는 생성자를 작성합니다. `GetArea()` 메서드에서 `Math.PI`를 사용하여 면적을 계산하고 반환합니다. 반지름 10과 5인 원의 면적을 소수점 2자리까지 출력합니다.

**실행 결과**

```
반지름 10인 원의 면적: 314.16
반지름 5인 원의 면적: 78.54
```

---

## 과제 6: 생성자 오버로딩

`ConstructorLog` 클래스에 매개변수 없는 생성자와 문자열 매개변수가 있는 생성자를 작성합니다. 각 생성자에서 어떤 생성자가 실행되었는지 메시지를 출력합니다.

**실행 결과**

```
기본 생성자 실행
오버로드된 생성자 실행: C#
오버로드된 생성자 실행: Unity
```

## 과제 7: 생성자 오버로딩 활용

`Person` 클래스에 3개의 생성자를 작성합니다. 매개변수 없는 생성자(기본값 사용), 이름만 받는 생성자, 이름과 나이를 모두 받는 생성자를 구현합니다.

**실행 결과**

```
이름: 홍길동, 나이: 21
이름: 백두산, 나이: 21
이름: 임꺽정, 나이: 30
```

---

## 과제 8: this() 생성자로 다른 생성자 호출

`Say` 클래스에서 `this()`를 사용하여 매개변수 있는 생성자가 먼저 기본 생성자를 호출하도록 합니다. 기본 생성자에서 "안녕하세요." 메시지를, 매개변수 생성자에서 전달받은 메시지를 출력합니다.

**실행 결과**

```
[1] 안녕하세요.
[2] 잘가요.
```

---

## 과제 9: 생성자 전달 (Constructor Forwarding)

`Money` 클래스에서 기본 생성자가 `this(1000)`으로 다른 생성자를 호출합니다. 매개변수 생성자에서 금액을 천 단위 구분자와 함께 출력합니다.

**실행 결과**

```
Money: 1,000
Money: 2,000
```

---

## 과제 10: 정적 생성자와 인스턴스 생성자

`Person` 클래스에 정적 생성자와 인스턴스 생성자를 모두 작성합니다. 정적 생성자에서 `_name` 정적 필드를 초기화하고, 인스턴스 생성자에서 `_age` 필드를 초기화합니다. 정적 생성자가 한 번만 호출되는 것을 확인합니다.

**실행 결과**

```
정적 생성자 호출됨
이름: 홍길동
나이: 21
나이: 30
```

---

## 과제 11: 읽기 전용 필드와 생성자

`WhichService` 클래스에 `readonly` 필드 `_serviceName`을 선언합니다. 생성자에서만 값을 할당하고 `Run()` 메서드에서 서비스 이름과 함께 실행 메시지를 출력합니다.

**실행 결과**

```
[1] 파일 로그 기능을 실행합니다.
[2] DB 로그 기능을 실행합니다.
```

## 과제 12: 식 본문 생성자

`Pet` 클래스에 화살표 연산자(`=>`)를 사용한 식 본문 생성자를 작성합니다. `ToString()` 메서드를 오버라이드하여 이름을 반환합니다.

**실행 결과**

```
야옹이
```
