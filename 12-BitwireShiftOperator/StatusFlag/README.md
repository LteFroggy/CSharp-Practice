# 캐릭터 상태 플래그 시뮬레이션

게임 캐릭터의 상태를 비트 플래그로 관리하는 시뮬레이션 프로그램을 작성하세요.

**요구사항**

- 4가지 상태 효과 정의: 독(1), 화상(2), 빙결(4), 기절(8)
- 고정된 시나리오를 순차적으로 실행
  1. 독 상태 추가
  2. 화상 상태 추가
  3. 빙결 상태 확인
  4. 독 상태 확인
  5. 독 상태 제거
- 각 단계에서 연산 과정과 결과를 상세히 출력

## 예상 실행 결과

```
=== 캐릭터 상태 플래그 시뮬레이션 ===

초기 상태: 0b0000 (0)

--- 1. 독 상태 추가 ---
연산: playerStatus |= k_Poison
계산: 0 | 1 = 1
결과: 0b0001 (1)

--- 2. 화상 상태 추가 ---
연산: playerStatus |= k_Burning
계산: 1 | 2 = 3
결과: 0b0011 (3)

--- 3. 빙결 상태 확인 ---
연산: (playerStatus & k_Frozen) != 0
계산: 3 & 4 = 0
결과: False (빙결 상태 아님)

--- 4. 독 상태 확인 ---
연산: (playerStatus & k_Poison) != 0
계산: 3 & 1 = 1
결과: True (독 상태임)

--- 5. 독 상태 제거 ---
연산: playerStatus &= ~k_Poison
~k_Poison = ~1 = -2 (2진수: ...1110)
계산: 3 & -2 = 2
결과: 0b0010 (2)

=== 최종 상태 ===
playerStatus: 0b0010 (2)
  - 독:   False
  - 화상: True
  - 빙결: False
  - 기절: False
```
