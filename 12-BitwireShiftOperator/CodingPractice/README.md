# 코딩 연습 - 비트 연산자와 시프트 연산자

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 이진수와 비트

### 1-1. 이진수 리터럴

`0b` 접두사를 사용하여 이진수 리터럴로 정수 변수를 선언하고 출력하세요. 언더스코어 구분자(`_`)를 사용한 이진수 리터럴도 선언하여 출력하세요.

**실행 결과**

```
10
10
```

### 1-2. 10진수와 2진수 변환

`Convert.ToString()` 메서드를 사용하여 10진수 `10`을 2진수 문자열로 변환하여 출력하고, `Convert.ToInt32()` 메서드를 사용하여 2진수 문자열 `"1010"`을 10진수로 변환하여 출력하세요.

**실행 결과**

```
10진수 10 = 2진수 1010
2진수 1010 = 10진수 10
```

## 2. 비트 연산자

### 2-1. 비트 AND 연산자 (&)

`byte` 타입 변수 `x`에 `0b1010`, `y`에 `0b1100`을 저장하고, AND 연산(`x & y`) 결과를 이진수와 10진수로 출력하세요.

**실행 결과**

```
  1010 (10)
& 1100 (12)
------
  1000 (8)
```

### 2-2. 비트 OR 연산자 (|)

`byte` 타입 변수 `x`에 `0b1010`, `y`에 `0b1100`을 저장하고, OR 연산(`x | y`) 결과를 이진수와 10진수로 출력하세요.

**실행 결과**

```
  1010 (10)
| 1100 (12)
------
  1110 (14)
```

### 2-3. 비트 XOR 연산자 (^)

`byte` 타입 변수 `x`에 `0b1010`, `y`에 `0b1100`을 저장하고, XOR 연산(`x ^ y`) 결과를 이진수와 10진수로 출력하세요.

**실행 결과**

```
  1010 (10)
^ 1100 (12)
------
  0110 (6)
```

### 2-4. 비트 NOT 연산자 (~)

`byte` 타입 변수 `x`에 `0b0000_1010`을 저장하고, NOT 연산(`~x`) 결과를 출력하세요. `int`로 변환한 결과와 `byte`로 캐스팅한 결과를 각각 출력하세요.

**실행 결과**

```
~ 00001010 (10)
----------
  결과값: -11
  11110101 (245)
```

### 2-5. 비트 연산자 종합

`int` 타입 변수 `x`에 `0b1010`, `y`에 `0b0110`을 저장하고, AND, OR, XOR, NOT 연산 결과를 모두 출력하세요.

**실행 결과**

```
x = 10 (0b1010)
y = 6 (0b0110)

x & y = 2 (0b0010)
x | y = 14 (0b1110)
x ^ y = 12 (0b1100)
~x = -11
```

## 3. 시프트 연산자

### 3-1. 왼쪽 시프트 연산자 (<<)

정수 `2`를 왼쪽으로 1비트, 2비트, 3비트 시프트한 결과를 이진수와 10진수로 출력하세요.

**실행 결과**

```
원본: 2 (0b0010)

2 << 1 = 4 (0b0100)  // × 2
2 << 2 = 8 (0b1000)  // × 4
2 << 3 = 16 (0b00010000)  // × 8
```

### 3-2. 오른쪽 시프트 연산자 (>>)

정수 `24`를 오른쪽으로 1비트, 2비트, 3비트 시프트한 결과를 이진수와 10진수로 출력하세요.

**실행 결과**

```
원본: 24 (0b00011000)

24 >> 1 = 12 (0b00001100)  // ÷ 2
24 >> 2 = 6 (0b00000110)  // ÷ 4
24 >> 3 = 3 (0b00000011)  // ÷ 8
```

### 3-3. 시프트 연산 기본

정수 `2`를 왼쪽으로 1비트 시프트, 오른쪽으로 1비트 시프트한 결과를 각각 출력하세요.

**실행 결과**

```
원본: 2
왼쪽 시프트 (× 2): 4
오른쪽 시프트 (÷ 2): 1
```

## 4. 복합 할당 연산자

`byte` 타입 변수 5개를 모두 `4`로 초기화하고, 각각 `&=`, `|=`, `^=`, `<<=`, `>>=` 복합 할당 연산자를 적용한 결과를 출력하세요.

**실행 결과**

```
4 &= 5  결과: 4
4 |= 1  결과: 5
4 ^= 2  결과: 6
4 <<= 1 결과: 8
4 >>= 1 결과: 2
```

## 5. 연산자 우선순위

산술 연산자와 괄호를 사용한 연산자 우선순위 예제를 작성하세요. `3 + 4 * 2`, `(3 + 4) * 2`, `10 / 5 * 2 + 1`, `15 / (5 * (2 + 1))`의 결과를 각각 출력하세요.

**실행 결과**

```
11
14
5
1
```

## 6. 비트 연산 활용 사례

### 6-1. 플래그 관리

상태 플래그 상수를 정의하고(독: `0b0001`, 화상: `0b0010`, 빙결: `0b0100`, 기절: `0b1000`), OR 연산으로 상태를 추가하고, AND 연산으로 상태를 확인하고, AND NOT 연산으로 상태를 제거하는 코드를 작성하세요.

**실행 결과**

```
현재 상태: 0011
독 상태: True
빙결 상태: False
독 해제 후: 0010
```

### 6-2. XOR을 이용한 값 교환

`int` 타입 변수 `a`에 `10`, `b`에 `20`을 저장하고, 임시 변수 없이 XOR 연산만으로 두 값을 교환하세요.

**실행 결과**

```
교환 전: a = 10, b = 20
교환 후: a = 20, b = 10
```

### 6-3. 홀수/짝수 판별

비트 AND 연산(`i & 1`)을 사용하여 1부터 5까지의 숫자가 홀수인지 짝수인지 판별하여 출력하세요.

**실행 결과**

```
1은(는) 홀수
2은(는) 짝수
3은(는) 홀수
4은(는) 짝수
5은(는) 홀수
```
