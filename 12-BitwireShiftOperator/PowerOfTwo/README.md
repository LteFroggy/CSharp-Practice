# 2의 거듭제곱 계산기

시프트 연산자를 사용하여 2의 거듭제곱을 계산하는 프로그램을 작성하세요.

**요구사항**

- 2^0 부터 2^10 까지의 값을 시프트 연산자로 계산하여 출력
- 왼쪽 시프트 연산자(`<<`)를 사용
- 지수와 결과값을 정렬하여 출력

## 예상 실행 결과

```
--- 2의 거듭제곱 (시프트 연산) ---
2^0  = 1
2^1  = 2
2^2  = 4
2^3  = 8
2^4  = 16
2^5  = 32
2^6  = 64
2^7  = 128
2^8  = 256
2^9  = 512
2^10 = 1024
```
