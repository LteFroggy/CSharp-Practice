# XOR 값 교환

XOR 연산을 사용하여 임시 변수 없이 두 정수의 값을 교환하는 프로그램을 작성하세요.

**요구사항**

- 사용자로부터 두 정수를 입력받음
- 임시 변수를 사용하지 않고 XOR 연산만으로 두 값을 교환
- 교환 전후의 값을 출력
- 각 XOR 연산 단계별 중간 결과도 출력하여 원리를 보여줌

## 예상 실행 결과

```
=== XOR 값 교환 ===

첫 번째 정수: 15
두 번째 정수: 27

교환 전: a = 15, b = 27

--- 교환 과정 ---
1단계: a = a ^ b
       a = 15 ^ 27 = 20

2단계: b = b ^ a
       b = 27 ^ 20 = 15

3단계: a = a ^ b
       a = 20 ^ 15 = 27

교환 후: a = 27, b = 15
```
