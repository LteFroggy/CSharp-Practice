# 증감 연산자 예측

다음 코드를 작성하고, 각 줄의 실행 후 변수 값을 출력하여 전위/후위 연산자의 동작을 확인하는 프로그램을 작성하세요.

1. `int x = 5;` 선언
2. `int a = x++;` 실행 후 `x`와 `a` 값 출력
3. `int b = ++x;` 실행 후 `x`와 `b` 값 출력
4. `int c = x--;` 실행 후 `x`와 `c` 값 출력
5. `int d = --x;` 실행 후 `x`와 `d` 값 출력

각 단계에서 "연산 후"라는 설명과 함께 관련 변수들의 값을 출력해야 합니다.

## 예상 실행 결과

```
초기 x: 5
x++ 후: x = 6, a = 5
++x 후: x = 7, b = 7
x-- 후: x = 6, c = 7
--x 후: x = 5, d = 5
```
