# 코딩 연습 - 할당 연산자와 증감 연산자

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 할당 연산자

### 1-1. 기본 할당 연산자

`var`를 사용하여 문자열 변수 `name`에 `"C#"`을, 실수 변수 `version`에 `10.0`을 저장한 후, 문자열 보간을 사용하여 두 값을 함께 출력하세요.

**실행 결과**

```
C# 10
```

### 1-2. 할당 연산자의 동작 원리

정수 변수 `num`에 `10`을 저장하고, `num + 5`의 결과를 정수 변수 `result`에 저장한 후, 두 변수의 값을 각각 출력하세요.

**실행 결과**

```
num: 10
result: 15
```

### 1-3. 변수 값 교환하기 (스왑)

정수 변수 `i`에 `100`, `j`에 `200`을 저장하고, 교환 전 값을 출력하세요. 임시 변수 `temp`를 사용하여 두 변수의 값을 교환한 후, 교환 후 값을 출력하세요.

**실행 결과**

```
교환 전: i = 100, j = 200
교환 후: i = 200, j = 100
```

## 2. 복합 할당 연산자

### 2-1. 기본형과 축약형 비교

정수 변수 `x`와 `y`를 각각 `3`으로 초기화하세요. `x`에는 기본형(`x = x + 2`)으로, `y`에는 축약형(`y += 2`)으로 2를 더한 후, 두 변수의 값을 출력하세요.

**실행 결과**

```
x: 5, y: 5
```

### 2-2. 복합 할당 연산자 활용

정수 변수 `a`를 `10`으로 초기화하고 초기값을 출력하세요. 이후 `+= 5`, `-= 3`, `*= 2`, `/= 4`, `%= 3`을 순서대로 적용하며 각 단계의 결과를 출력하세요.

**실행 결과**

```
초기값: 10
a += 5 결과: 15
a -= 3 결과: 12
a *= 2 결과: 24
a /= 4 결과: 6
a %= 3 결과: 0
```

### 2-3. 누적 연산 예제

정수 변수 `total`을 `0`으로 초기화한 후, `+= 10`, `+= 20`, `+= 30`을 순서대로 누적하고 최종 총합을 출력하세요.

**실행 결과**

```
총합: 60
```

## 3. 증감 연산자

### 3-1. 증가 연산자

정수 변수 `num`을 `10`으로 초기화하세요. 기본형(`num = num + 1`), 복합 할당(`num += 1`), 증가 연산자(`num++`)를 각각 한 번씩 적용하며 매번 값을 출력하세요.

**실행 결과**

```
11
12
13
```

### 3-2. 감소 연산자

정수 변수 `num`을 `10`으로 초기화하세요. 기본형(`num = num - 1`), 복합 할당(`num -= 1`), 감소 연산자(`num--`)를 각각 한 번씩 적용하며 매번 값을 출력하세요.

**실행 결과**

```
9
8
7
```

### 3-3. 전위 연산자 예제

정수 변수 `i`를 `3`으로 초기화하고, `int j = ++i;`를 실행한 후 `i`와 `j`의 값을 출력하세요.

**실행 결과**

```
i: 4
j: 4
```

### 3-4. 후위 연산자 예제

정수 변수 `x`를 `3`으로 초기화하고, `int y = x++;`를 실행한 후 `x`와 `y`의 값을 출력하세요.

**실행 결과**

```
x: 4
y: 3
```

### 3-5. 전위와 후위 비교 예제

정수 변수 `a`와 `b`를 각각 `5`로 초기화하세요. `++a`를 `Console.WriteLine`에서 직접 출력하고 이어서 `a`의 값을 출력하세요. 마찬가지로 `b++`를 `Console.WriteLine`에서 직접 출력하고 이어서 `b`의 값을 출력하세요.

**실행 결과**

```
전위 ++a: 6
a의 값: 6
후위 b++: 5
b의 값: 6
```

### 3-6. 증감 연산자 종합 예제

정수 변수 `number1 = 3`, `number2 = 5`, `result1 = 0`, `result2 = 0`을 선언하세요. `number1--`, `result1 = ++number1`, `++number2`, `result2 = number2--`를 순서대로 실행한 후 네 변수의 값을 모두 출력하세요.

**실행 결과**

```
result1: 3
result2: 6
number1: 3
number2: 5
```
