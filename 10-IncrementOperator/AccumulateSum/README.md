# 누적 합계 계산

정수 변수 `sum`과 `count`를 사용하여 5개의 숫자(10, 20, 30, 40, 50)를 누적 합산하는 프로그램을 작성하세요.

요구사항:
- `sum` 변수에 각 숫자를 `+=` 연산자로 누적
- `count` 변수에 `++` 연산자로 카운트 증가
- 각 숫자를 더할 때마다 현재 카운트와 누적 합계를 출력
- 마지막에 총 개수와 최종 합계를 출력

## 예상 실행 결과

```
1번째 숫자 10 추가: 합계 = 10
2번째 숫자 20 추가: 합계 = 30
3번째 숫자 30 추가: 합계 = 60
4번째 숫자 40 추가: 합계 = 100
5번째 숫자 50 추가: 합계 = 150
---
총 5개 숫자의 합계: 150
```
