# 행렬 대각선 합계

3x3 정수 행렬의 대각선 요소를 분석하는 프로그램을 작성하세요.

### 요구사항

1. 3x3 2차원 정수 배열을 다음 값으로 초기화하세요:

   ```
   1  2  3
   4  5  6
   7  8  9
   ```

2. 다음 기능을 구현하세요:
   - **행렬 출력**: 2차원 배열을 표 형태로 출력
   - **주대각선 합계**: 왼쪽 위에서 오른쪽 아래 방향 (1, 5, 9)
   - **부대각선 합계**: 오른쪽 위에서 왼쪽 아래 방향 (3, 5, 7)
   - **대각선 교차점**: 두 대각선이 만나는 중앙 요소 출력

## 예상 실행 결과

```
=== 행렬 대각선 합계 ===

[행렬 출력]
1  2  3
4  5  6
7  8  9

[대각선 분석]
주대각선 합계: 15
부대각선 합계: 15
대각선 교차점: 5
```
