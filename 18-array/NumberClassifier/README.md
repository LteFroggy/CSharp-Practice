# 숫자 분류기

정수 배열의 요소들을 다양한 기준으로 분류하는 프로그램을 작성하세요.

### 요구사항

1. 정수 배열을 다음 값으로 초기화하세요:
   - -5, 12, 0, -8, 7, 15, -3, 20, 9, -1

2. 다음 기능을 구현하세요:
   - **원본 배열 출력**: 모든 요소를 쉼표로 구분하여 출력
   - **양수/음수/영 분류**: 각 카테고리에 해당하는 숫자들을 출력하고 개수를 표시
   - **홀수/짝수 분류**: 각 카테고리에 해당하는 숫자들을 출력하고 개수를 표시
   - **절댓값 10 이상**: 절댓값이 10 이상인 숫자들을 출력

## 예상 실행 결과

```
=== 숫자 분류기 ===

[원본 배열]
-5, 12, 0, -8, 7, 15, -3, 20, 9, -1

[양수/음수/영 분류]
양수 (5개): 12, 7, 15, 20, 9
음수 (4개): -5, -8, -3, -1
영 (1개): 0

[홀수/짝수 분류]
홀수 (6개): -5, 7, 15, -3, 9, -1
짝수 (4개): 12, 0, -8, 20

[절댓값 10 이상]
12, 15, 20
```
