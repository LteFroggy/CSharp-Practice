# 배열 검색

정수 배열에서 특정 값을 찾아 위치를 출력하는 프로그램을 작성하세요.

### 요구사항

1. 정수 배열을 다음 값으로 초기화하세요:
   - 15, 8, 23, 42, 16, 8, 31

2. 다음 기능을 구현하세요:
   - **배열 출력**: 인덱스와 함께 모든 요소 출력
   - **값 검색 (첫 번째 위치)**: 값 8을 찾아 처음 발견된 위치(인덱스) 출력
   - **값 검색 (모든 위치)**: 값 8이 있는 모든 위치 출력
   - **없는 값 검색**: 값 100을 찾고, 없으면 "찾을 수 없음" 출력

## 예상 실행 결과

```
=== 배열 검색 프로그램 ===

[배열 내용]
[0]=15  [1]=8  [2]=23  [3]=42  [4]=16  [5]=8  [6]=31

[값 8 검색 - 첫 번째 위치]
값 8을 인덱스 1에서 찾았습니다.

[값 8 검색 - 모든 위치]
값 8이 있는 위치: 1, 5
총 2개 발견

[값 100 검색]
값 100을 찾을 수 없습니다.
```
