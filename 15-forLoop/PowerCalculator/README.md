# 거듭제곱 계산기

사용자로부터 밑(base)과 지수(exponent)를 입력받아 거듭제곱을 계산하는 프로그램을 작성하세요.

**요구사항**

- "밑을 입력하세요: "와 "지수를 입력하세요: " 메시지로 각각 정수를 입력 받음
- for 문을 사용하여 밑을 지수만큼 반복 곱하여 거듭제곱 계산
- 결과를 "A의 B제곱: C" 형식으로 출력

## 예상 실행 결과

```
밑을 입력하세요: 2
지수를 입력하세요: 10
2의 10제곱: 1024
```

```
밑을 입력하세요: 3
지수를 입력하세요: 4
3의 4제곱: 81
```

```
밑을 입력하세요: 5
지수를 입력하세요: 0
5의 0제곱: 1
```
