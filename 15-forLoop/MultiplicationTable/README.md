# 구구단 출력

사용자로부터 시작 단과 끝 단을 입력받아 해당 범위의 구구단을 가로로 출력하는 프로그램을 작성하세요.

**요구사항**

- "시작 단을 입력하세요: "와 "끝 단을 입력하세요: " 메시지로 입력 받음
- 각 단의 헤더를 먼저 출력 (예: "2단  3단  4단")
- 곱셈 결과를 가로로 정렬하여 출력
- 결과값이 한 자리일 때 공백으로 정렬 (예: `{결과,2}`)

## 예상 실행 결과

```
시작 단을 입력하세요: 2
끝 단을 입력하세요: 4
  2단    3단    4단
2x1= 2  3x1= 3  4x1= 4
2x2= 4  3x2= 6  4x2= 8
2x3= 6  3x3= 9  4x3=12
2x4= 8  3x4=12  4x4=16
2x5=10  3x5=15  4x5=20
2x6=12  3x6=18  4x6=24
2x7=14  3x7=21  4x7=28
2x8=16  3x8=24  4x8=32
2x9=18  3x9=27  4x9=36
```
