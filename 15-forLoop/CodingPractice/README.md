# 코딩 연습 - 반복문 for 문

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 첫 번째 for 문 예제

`for` 문을 사용하여 `"안녕하세요."`를 3번 출력하세요.

**실행 결과**

```
안녕하세요.
안녕하세요.
안녕하세요.
```

---

## 2. 반복 변수의 활용

반복 변수 `i`를 1부터 5까지 증가시키면서 `"Count: {i}"` 형식으로 출력하세요.

**실행 결과**

```
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
```

---

## 3. 증감식 변형

### 3-1. 2씩 증가

`for` 문에서 반복 변수를 0부터 시작하여 2씩 증가시키며 5 미만까지 출력하세요.

**실행 결과**

```
0
2
4
```

### 3-2. 감소하는 반복

`for` 문에서 반복 변수를 5부터 시작하여 1씩 감소시키며 0보다 클 때까지 출력하세요.

**실행 결과**

```
5
4
3
2
1
```

### 3-3. 증가와 감소 비교

`"증가:"`를 출력한 후 1부터 5까지 탭(`\t`)으로 구분하여 가로로 출력하고, `"감소:"`를 출력한 후 5부터 1까지 탭으로 구분하여 가로로 출력하세요.

**실행 결과**

```
증가:
1	2	3	4	5
감소:
5	4	3	2	1
```

---

## 4. 합계 계산

변수 `n`에 10을 저장하고, `for` 문을 사용하여 1부터 `n`까지의 합계를 계산한 후 결과를 출력하세요.

**실행 결과**

```
1부터 10까지의 합: 55
```

---

## 5. 조건을 활용한 반복

변수 `n`에 10을 저장하고, `for` 문 안에서 `if` 문을 사용하여 짝수만 합산한 결과를 출력하세요.

**실행 결과**

```
1부터 10까지 짝수의 합: 30
```

---

## 6. 중첩 for 문

### 6-1. 별 삼각형 출력

중첩 `for` 문을 사용하여 5줄의 직각삼각형 별 패턴을 출력하세요. 바깥 `for` 문은 행을, 안쪽 `for` 문은 각 행의 별 개수를 제어합니다.

**실행 결과**

```
*
**
***
****
*****
```

## 7. 팩토리얼 계산

중첩 `for` 문을 사용하여 1부터 5까지의 팩토리얼(`n!`)을 계산하고 출력하세요. 바깥 `for` 문은 `i`를 1부터 5까지, 안쪽 `for` 문은 1부터 `i`까지 곱하여 팩토리얼 값을 구합니다.

**실행 결과**

```
1! = 1
2! = 2
3! = 6
4! = 24
5! = 120
```

---

## 8. 무한 루프

`for ( ; ; )` 형태의 무한 루프를 사용하여 반복 횟수를 출력하고, 3번 반복 후 `break`로 탈출하세요.

**실행 결과**

```
반복 0
반복 1
반복 2
반복 종료
```

---

## 9. for 문의 구성 요소 생략

초기식을 `for` 문 바깥에서 선언하고, 증감식을 본문 안에서 실행하는 방식으로 0부터 2까지 출력하세요. `for` 문에는 조건식만 사용합니다.

**실행 결과**

```
0
1
2
```
